# Introduction

Ce document explique comment récupérer et utiliser les résultats stockés
dans les fichiers `.RData` dans votre script R.

## Charger les données

Pour charger le fichier `.RData` dans R, utilisez la fonction `load()` :

``` r
load("dada2_V2.RData")
```
